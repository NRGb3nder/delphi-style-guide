# Соглашения о наименовании

### Объявленные типы, классы

Идентификатор типа или класса должен начинаться с заглавной буквы `Т` \(от "type"\) после которой следует имя класса с заглавной буквы. Оформление необходимо выполнять согласно CamelCase.

Пример корректного идентификатора:

```Pascal
TMyClass
```

Исключениями из правил являются типы-указатели. Их названия всегда начинаются с заглавной буквы `P` \(от "pointer"\).
